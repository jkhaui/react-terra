# `useLiveBalances`

## API

- `showLuna: boolean`
  Optional
  Whether to show LUNA amongst a user's stablecoin balances.
- `disableCaching: boolean`
  Optional
  It is recommended that the default caching strategy is used, but this can
  be overridden.
- `cw20TokenList: CW20Token[]`
  Optional
  ~~An array of white-listed token strings~~ Not yet implemented
- `refetchInterval: number`
  Optional
  Defaults to the Terra block time (6 seconds).
